# Timetable Extractor

A lightweight Flask app that ingests a PDF timetable and surfaces each session with its time, subject, room, and teacher. After parsing, you can also download the extracted data as a CSV file.

## Quick start

1. **Create a virtual environment (recommended)**
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\activate
   ```
2. **Install dependencies**
   ```powershell
   pip install -r requirements.txt
   ```
3. **Run the server**
   ```powershell
   flask --app app run --debug
   ```
4. **Upload a timetable**
   - Open http://127.0.0.1:5000/
   - Type the class/section label you care about (e.g., `BSCS-6B`) so only matching rows are kept
   - Upload the PDF timetable (the parser is tuned for the official RSCI grid layout)
   - Review the extracted rows (Day, Time, Section, Subject, Room, Teacher) and download the CSV snapshot

## Notes

- The parser locks onto the grid by reading table borders. If your PDF removes the grid lines or uses a completely different layout, adjust `TABLE_SETTINGS`, slot timings, or the `resolve_slot_count` logic inside `app.py`.
- Filtering is substring-based. Make sure the class label you enter appears verbatim somewhere in each relevant line of the PDF.
- Uploaded files are processed in memory only. CSV exports are cached temporarily for download and automatically rotated to keep memory usage low.

## Deploy (Render)

1. Push this folder to a GitHub repo.
2. In Render, click **New +** → **Blueprint**.
3. Connect your repo and select it.
4. Render will auto-detect `render.yaml` and create the web service.
5. Wait for build/deploy, then open the generated URL.

### Environment variables

- `FLASK_SECRET_KEY` is auto-generated by `render.yaml`.
- Optional: set your own strong value manually in Render if preferred.

## Deploy (Railway/Heroku-style)

- This repo also includes a `Procfile` with:
   - `web: waitress-serve --listen=0.0.0.0:$PORT app:app`
- On platforms that support Procfiles, use that as the start process.

## Deploy (GitHub Pages - Free)

This repo now includes a static browser version in `docs/` for free hosting on GitHub Pages.

1. Push your latest `main` branch (already done in this project).
2. Open your repository on GitHub.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/docs`
5. Save and wait ~1–3 minutes.

Your site URL will be:

- `https://mustafa-exe.github.io/timetable-extractor/`

### GitHub Pages notes

- The Pages app is fully client-side (no Flask backend).
- PDF parsing, filtering, CSV export, and saved library run in-browser.
- Saved library is stored in `localStorage` for that browser/device only.
